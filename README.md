# JSON Item Counter

A powerful and user-friendly web application designed to analyze and count items within JSON structures.

## Features

- **Deep Analysis**: Counts total nodes, objects, arrays, strings, numbers, booleans, and null values.
- **Real-time Processing**: Paste your JSON and get instant results.
- **User-Friendly Interface**: Clean, responsive design with clear metrics.
- **Error Handling**: Validates JSON input and provides helpful error messages.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/JC-dev-stack/JSONCounter.git
   ```
2. Open `index.html` in your web browser.

## Usage

1. Paste your JSON data into the input area.
2. Click "Count Items" to analyze.
3. View the detailed breakdown of your JSON structure.

## Technologies

- HTML5
